# STT AI Editor - Module 027: GUI Final Workflow

Adds the recommended one-click GUI workflow:

`Run Final Wedding V2 + Live Review`

It will:

1. Run Wedding Pipeline V2
2. Automatically open Live Manual Review
3. Let you KEEP / REJECT
4. Save directly to project folder
5. Export Latest Manual XML from GUI
6. Open Latest XML Folder

Copy into:

`D:\Projects\STT-AI-Editor`

Run:

```powershell
python scripts/run_gui.py
```

Use:

1. Click `Run Final Wedding V2 + Live Review`
2. Wait until browser opens
3. KEEP / REJECT
4. Click `Save to Project Folder`
5. Back to GUI: click `Export Latest Manual XML`
6. Click `Open Latest XML Folder`
7. Import `stt_ai_premiere_import.xml` into Premiere

This module keeps old buttons but makes the recommended workflow clearer at the top.
