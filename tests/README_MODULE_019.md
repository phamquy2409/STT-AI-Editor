# STT AI Editor - Module 019: GUI Project Creator

Copy files into:

`D:\Projects\STT-AI-Editor`

Run:

```powershell
python scripts/run_gui.py
```

New in Module 019:

- Create New Project directly in GUI
- Choose Projects Root
- Project Name field
- Set created project as Active Project automatically
- Open Existing Project button
- Open Project button

Recommended use for current project:

Do not tick `Run from scratch`.

Recommended use for a new wedding project:

1. Choose `Projects Root`, example:
   `D:\STT Projects`
2. Enter project name, example:
   `Wedding_Client_001`
3. Choose source folder
4. Click `Create New Project`
5. Tick `Run from scratch`
6. Click `Run Pipeline`
