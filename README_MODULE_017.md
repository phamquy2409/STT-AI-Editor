# STT AI Editor - Module 017: Simple GUI App

Copy files into:

`D:\Projects\STT-AI-Editor`

Run:

```powershell
python scripts/run_gui.py
```

GUI buttons:

- Chọn Project
- Chọn Source
- Run Pipeline
- Generate Manual Review
- Export Manual XML
- Open Exports
- Open Latest Review
- Open Latest Manual Review
- Open Latest XML Folder

Default paths:

- Project: `D:\STT Projects\Wedding_Test_001`
- Source: `D:\5thang5test`

Use:

1. Run Pipeline
2. App opens `manual_review.html`
3. Mark KEEP / REJECT
4. Export JSON
5. In GUI click `Export Manual XML`
6. Choose `manual_selection.json`
7. Import the new `stt_ai_premiere_import.xml` into Premiere

Do not tick `Run from scratch` for your current test project unless you want to scan/detect/analyze again.
